# INGREMATCH-SMART-COSMETIC-CHECKER

## 📌 PROJECT SUMMARY
IngreMatch is a UI-based web application developed to help users instantly explore ingredient details and availability across 1,470+ Sephora cosmetic products. The platform enables real-time product search by name, providing structured ingredient insights and improving transparency in cosmetic selection. The application focuses on transforming raw product and ingredient data into accessible, user-friendly insights. In addition to the web app interface, an interactive Tableau dashboard was developed to analyze product labels, pricing patterns and ingredient distribution trends.

## 🎯 Key Objectives
1. Enable real-time ingredient lookup for cosmetic products
2. Improve product transparency through structured data analysis
3. Visualize pricing trends and ingredient frequency distribution
4. Provide an intuitive UI for seamless product search and exploration

## 🔍 Core Features
1. Instant product search functionality
2. Detailed ingredient breakdown per product
3. Interactive Tableau dashboard for trend analysis
4. Price trend and label distribution visualization
5. Ingredient frequency and category analysis

## 📊 Analytical Insights Covered
1. Ingredient distribution patterns
2. Clean-label and formulation trends
3. Product pricing comparison
4. Brand-wise and category-wise segmentation
5. Correlation between price and formulation complexity
